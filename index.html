Arduino and Unity

The main idea of this experiment was to check how would Arduino work as a device to move a cube in the Unity scene. Used components: Arduino Nano and an ultrasound sensor. The ultrasound sensor was used to measure the distance to the nearest obstacle and depending on the measurement, the cube moved faster or slower (when the distance was shorter, the cube moved slower, and it moved faster when the distance was greater).

About

Hello there! My name is Maja Mrozińska and I'm a student of Automatic Control and Robotics with a specialization in Robots and Autonomous Systems at Poznan University of Technology. This portfolio is a representation of my recent projects in Unity and C#. I'm a self-taught and I would love to countinuously improve my skill in this area. My passion for gaming started when I got my first console, which was an Atari obviously. Racing games became my favourite, so I've spent a lot of time playing games like Need For Speed or WipeOut later. I also love mobile games and I would love to develop a completely new one someday. I've spent so much time playing random games that one day I thought that maybe I should start making them. So I started my first game in python. It was a disaster. But I didn't give up. Later I read about Unity engine and I loved it. I had learnt some C# in the past so I was happy that I was familiar with it when I started scripting. It was really exciting, everyday I learn more and more about what is Unity capable of. Despite the fact, that I've been working with Unity only for a couple of months, I'm totally sure I would like to become a true Unity Developer one day.
